# F6.T5: Blazor Dashboard

## Agent: FULLSTACK_BLAZOR

## Task Overview
Create a web dashboard for monitoring training progress, visualizing statistics, and controlling training.

## Context
Create new project `src/TzarBot.Dashboard/`. Use Blazor Server with SignalR.

## Requirements

1. Create Blazor Server project:
   ```bash
   dotnet new blazorserver -n TzarBot.Dashboard
   ```

2. Create SignalR Hub:
   ```csharp
   public class TrainingHub : Hub
   {
       public async Task JoinMonitoring();
       public async Task LeaveMonitoring();
   }
   ```

3. Create `TrainingStateService`:
   - Inject ITrainingPipeline
   - Forward events via SignalR
   - Expose pause/resume controls

4. Create pages:
   - **Index.razor**: Main dashboard with status, stats grid, controls
   - **Generations.razor**: Generation history table
   - **Population.razor**: Current population viewer
   - **Settings.razor**: Training configuration

5. Add Chart.js integration:
   - Fitness over generations
   - Win rate chart
   - Population diversity

6. Real-time updates:
   - SignalR connection
   - State change notifications
   - Auto-refresh charts
