# Task F1.T1: Project Setup

## Task ID
`F1.T1`

## Description
Create the .NET 8 solution structure with all required projects and NuGet packages.

## Prerequisites
- None (this is the first task)

## Context Files to Read
- `plans/1general_plan.md` - for technology stack reference
- `plans/2_implementation_workflow.md` - for directory structure

## Instructions

Create the .NET 8 solution structure for TzarBot project.

### Step 1: Create Solution

```bash
dotnet new sln -n TzarBot
```

### Step 2: Create Projects

```bash
# Game Interface library
dotnet new classlib -n TzarBot.GameInterface -o src/TzarBot.GameInterface -f net8.0
dotnet sln add src/TzarBot.GameInterface/TzarBot.GameInterface.csproj

# Common library
dotnet new classlib -n TzarBot.Common -o src/TzarBot.Common -f net8.0
dotnet sln add src/TzarBot.Common/TzarBot.Common.csproj

# Test project
dotnet new xunit -n TzarBot.Tests -o tests/TzarBot.Tests -f net8.0
dotnet sln add tests/TzarBot.Tests/TzarBot.Tests.csproj
```

### Step 3: Add NuGet Packages

For `TzarBot.GameInterface`:
```bash
cd src/TzarBot.GameInterface
dotnet add package Vortice.Direct3D11
dotnet add package Vortice.DXGI
dotnet add package OpenCvSharp4
dotnet add package OpenCvSharp4.runtime.win
dotnet add package MessagePack
```

For `TzarBot.Tests`:
```bash
cd tests/TzarBot.Tests
dotnet add package FluentAssertions
dotnet add package Moq
```

### Step 4: Add Project References

```bash
dotnet add src/TzarBot.GameInterface/TzarBot.GameInterface.csproj reference src/TzarBot.Common/TzarBot.Common.csproj
dotnet add tests/TzarBot.Tests/TzarBot.Tests.csproj reference src/TzarBot.GameInterface/TzarBot.GameInterface.csproj
dotnet add tests/TzarBot.Tests/TzarBot.Tests.csproj reference src/TzarBot.Common/TzarBot.Common.csproj
```

### Step 5: Create Directory.Build.props

Create `Directory.Build.props` in root with:
```xml
<Project>
  <PropertyGroup>
    <TargetFramework>net8.0</TargetFramework>
    <Nullable>enable</Nullable>
    <ImplicitUsings>enable</ImplicitUsings>
    <TreatWarningsAsErrors>true</TreatWarningsAsErrors>
    <LangVersion>latest</LangVersion>
  </PropertyGroup>
</Project>
```

### Step 6: Create .editorconfig

Create `.editorconfig` with C# coding standards.

### Step 7: Create Placeholder Classes

In `TzarBot.Common`:
- `Models/ScreenFrame.cs`
- `Models/GameAction.cs`

## Expected Output Files
- `TzarBot.sln`
- `src/TzarBot.GameInterface/TzarBot.GameInterface.csproj`
- `src/TzarBot.Common/TzarBot.Common.csproj`
- `tests/TzarBot.Tests/TzarBot.Tests.csproj`
- `Directory.Build.props`
- `.editorconfig`

## Validation
Run after completion:
```bash
dotnet build TzarBot.sln
```

Expected: Build succeeds with no errors.

## On Failure
If build fails:
1. Check NuGet package versions are compatible with .NET 8
2. Verify Vortice.Windows supports your Windows version
3. Check for missing runtime dependencies
